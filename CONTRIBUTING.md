# Contributing

Contributions to this repository are very welcome. If you are interested in contributing, feel free to create an issue in the [issue tracking system](https://github.com/HicResearch/MELODY/issues). Alternatively, you may [fork](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/working-with-forks/fork-a-repo) the project and submit a [pull request](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-a-pull-request-from-a-fork). Please create an issue before starting any significant work so that we can discuss and understand the changes.
